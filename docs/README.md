# Linked

Gestor de links desarrollado en Android con Kotlin y Jetpack Compose.

## Descripción

Aplicación para gestionar enlaces con autenticación mediante Google Sign-In y backend en SpringBoot + PostgreSQL.

## Estructura del proyecto

- `docs/` - Documentación del proyecto
- `assets/` - Imágenes, iconos e ilustraciones
- `app/` - Código fuente de la aplicación Android

## Stack tecnológico

- **Frontend**: Android, Kotlin, Jetpack Compose, Material 3
- **Autenticación**: Google Sign-In (Credential Manager)
- **Red**: Retrofit (futuro)
- **Arquitectura**: MVVM
- **Backend**: SpringBoot + PostgreSQL (futuro)
